## Описание

Лабораторная работа посвящена методу наименьших квадратов и его применениям в задачах линейной регрессии. Реализовано вручную через NumPy без использования готовых регрессионных моделей.

## Зависимости

```
numpy
matplotlib
sympy
sklearn
```

## Данные

- `train.txt`, `test.txt` — одномерные данные для задания 1
- `flats_moscow_mod.txt` — датасет квартир для задания 2
